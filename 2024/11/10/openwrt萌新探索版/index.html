openwrt 萌新探索

本教材实现目标： 让萌新像使用普通路由器那样操作软路由（无脑点点点啦~）

0x01:需要准备的软件

  • 烧入软件：https://www.balena.io/etcher

  • 固件

    • 自选固件https://firmware-selector.openwrt.org/
    • 所有固件: https://downloads.openwrt.org/releases/23.05.5/targets/

  • 阿里源：（只要改host：https://mirrors.aliyun.com/openwrt）

  • src/gz openwrt_core https://mirrors.aliyun.com/openwrt/releases/23.05.5/targets/x86/64/packages
src/gz openwrt_base https://mirrors.aliyun.com/openwrt/releases/23.05.5/packages/x86_64/base
src/gz openwrt_luci https://mirrors.aliyun.com/openwrt/releases/23.05.5/packages/x86_64/luci
src/gz openwrt_packages https://mirrors.aliyun.com/openwrt/releases/23.05.5/packages/x86_64/packages
src/gz openwrt_routing https://mirrors.aliyun.com/openwrt/releases/23.05.5/packages/x86_64/routing
src/gz openwrt_telephony https://mirrors.aliyun.com/openwrt/releases/23.05.5/packages/x86_64/telephony

  • 挂载分区

  • 在 Luci 面板中操作：系统 –> 挂载点 –> 生成配置
然后往下拉，在 “挂载点” 中找到刚才新建的分区，点击修改
在 “启动此挂载点” 后面打上勾，“挂载点” 中选择 “作为根文件系统使用”，然后复制最后列出的几条命令备用，保存并应用
###
在openwrt终端执行
cd /
mkdir -p /tmp/introot
mkdir -p /tmp/extroot
mount --bind / /tmp/introot
mount /dev/sda3 /tmp/extroot # 你的可能不是 sda3，具体名称以cfdisk为准，就是新建的分区
tar -C /tmp/introot -cvf - . | tar -C /tmp/extroot -xf -
umount /tmp/introot
umount /tmp/extroot
#执行不报错 ，就重启

  • 其他一些常用插件

  • 1. 网络相关插件
luci-app-openvpn：OpenVPN 客户端和服务端的 Web 管理界面。
luci-app-wireguard：WireGuard VPN 的 Web 管理界面。
dnsmasq：轻量级的 DNS 和 DHCP 服务器，通常默认安装，用于局域网内部的 DNS 和 IP 地址分配。
luci-app-sqm：Smart Queue Management，帮助优化网络延迟和带宽管理，适用于家庭网络或有高流量需求的场景。
luci-app-vpnbypass：使某些设备或流量绕过 VPN 通道，适合有特定需求的用户。

2. 安全相关插件
luci-app-firewall：OpenWRT 的防火墙设置插件，提供友好的 Web 界面管理防火墙规则。
snort：一个开源的入侵检测系统（IDS），用于实时监控网络流量并分析是否存在恶意活动。
suricata：类似于 Snort，是一个高性能的入侵检测系统，支持多种协议的检测。
ipset：增强的 IP 过滤功能，允许通过 IP 集合（如黑名单或白名单）管理防火墙规则。

3. 存储和文件系统
luci-app-samba：用于配置 Samba 文件共享服务，允许 OpenWRT 路由器和局域网内的设备共享文件。
luci-app-ftp：提供 FTP 服务的插件，用于文件传输。
block-mount：挂载外部存储设备（如 USB 硬盘或 U 盘）到 OpenWRT 路由器上，通常需要配合 ext4、ntfs-3g 等文件系统支持包。

4. 带宽管理和流量监控
luci-app-nlbwmon：网络带宽监控插件，用于查看每个客户端的带宽使用情况。
luci-app-wrtbwmon：另一种流量监控工具，提供带宽使用分析、流量统计等。
tc：控制流量的 Linux 命令，结合 luci-app-sqm 使用，可以更好地管理网络延迟。

5. 系统优化和管理
luci-app-commands：允许用户通过 Web 界面执行自定义的 shell 命令。
luci-app-upnp：用于配置 UPnP (Universal Plug and Play) 服务，便于设备自动开放端口。
luci-app-autoreboot：允许用户设置路由器定期自动重启。
luci-app-ttyd：提供基于 Web 的终端访问，通过浏览器就可以连接到 OpenWRT 的命令行。

6. 远程访问和代理
luci-app-ssr-plus：ShadowsocksR 插件，用于科学上网。
luci-app-ss：Shadowsocks 插件，用于科学上网。
luci-app-trojan：Trojan 协议支持的插件，用于代理和科学上网。
luci-app-v2ray：V2Ray 插件，适用于搭建 V2Ray 代理服务。
7. 多媒体相关插件
luci-app-pptp：PPTP VPN 服务插件，用于提供 PPTP 服务。
minidlna：小型的 DLNA 媒体服务器，允许路由器在局域网内提供多媒体流服务。

8. 其他常见插件
luci-app-adblock：广告拦截插件，能够屏蔽广告和恶意网站。
luci-app-mjpeg-streamer：用于在 OpenWRT 上搭建 MJPEG 视频流服务。
luci-app-openclash：OpenClash 插件，提供 Clash 配置管理工具，广泛用于科学上网。

9. 定时任务和自动化
luci-app-cron：提供 Web 界面管理定时任务，支持定时执行命令或脚本。
luci-app-scheduled-reboot：定时重启路由器的插件。
10. 开发和调试工具
luci-app-logs：查看路由器日志的插件，便于调试和问题排查。
luci-app-statistics：提供流量和系统状态的图表，可帮助用户了解路由器性能。

  • 主题插件：搜theme 下载设置即可
