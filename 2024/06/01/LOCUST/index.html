​ 重要的属性：

1. 设置用户行为的时间间隔

  1. 固定时间间隔 

    1. from locust import HttpUser, task, between, constant
#通过 wait_time 设置一个固定的时间间隔。用户在每次任务之间等待相同的时间。
class MyUser(HttpUser):
    wait_time = constant(5)  # 每次任务之间等待固定的5秒

    @task
    def my_task(self):
        self.client.get("/path")

  2. 自定义时间间隔

  3. from locust import HttpUser, task
import random
#通过编写自定义的等待时间函数来定义更加复杂的等待时间逻辑。
def my_wait_time():
    return random.expovariate(1)  # 使用指数分布的随机时间

class MyUser(HttpUser):
    wait_time = my_wait_time  # 使用自定义的等待时间函数

    @task
    def my_task(self):
        self.client.get("/path")

  4. 使用多个任务函数

    1. from locust import HttpUser, task
#通过在 tasks 属性中定义一个列表，你可以让用户执行多个任务。任务之间会根据定义#的概率进行切换
class MyUser(HttpUser):
    @task
    def task_one(self):
        self.client.get("/path1")

    @task
    def task_two(self):
        self.client.get("/path2")

  5. 使用字典定义任务权重

    1. from locust import HttpUser, task
#可以使用字典来定义任务的权重，从而控制每个任务被执行的概率。权重值越大，任务被执行的频率越高
class MyUser(HttpUser):
    tasks = {
        "task_one": 3,  # 权重为3
        "task_two": 1   # 权重为1
    }

    @task
    def task_one(self):
        self.client.get("/path1")

    @task
    def task_two(self):
        self.client.get("/path2")

  6. 使用嵌套任务集合

    1. from locust import HttpUser, TaskSet, task
#你可以将任务集合嵌套起来，以创建更复杂的用户行为模式
class SubTaskSet(TaskSet):
    @task
    def sub_task(self):
        self.client.get("/subpath")

class MyTaskSet(TaskSet):
    tasks = [SubTaskSet]

    @task
    def task_one(self):
        self.client.get("/path1")

    @task
    def task_two(self):
        self.client.get("/path2")

class MyUser(HttpUser):
    tasks = [MyTaskSet]
